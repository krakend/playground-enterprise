Real-time bidirectional communication through KrakenD's WebSocket proxy. Messages are relayed between clients via the gateway.

Click Connect to join a chat room.

About this demo

/chat/ws/{room}

KrakenD proxies WebSocket connections to backend services, enabling real-time features while maintaining gateway-level control.

How it works

  1. Client opens WebSocket to ws://gateway:8080/chat/ws/{room}
  2. KrakenD upgrades the connection and proxies to the chat backend
  3. Messages flow bidirectionally through the gateway
  4. Open multiple browser tabs to see real-time messaging

Configuration highlights

{
  "endpoint": "/chat/ws/{room}",
  "input_headers": ["*"],
  "backend": [{
    "url_pattern": "/ws/{room}",
    "host": ["ws://chat:8888"],
    "encoding": "no-op",
    "extra_config": {
      "backend/websocket": {
        "input_headers": ["*"]
      }
    }
  }]
}
